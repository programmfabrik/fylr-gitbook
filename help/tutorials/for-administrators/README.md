# For Administrators

These tutorials are written for **administrators** who set up and operate a fylr instance. Where the reference sections describe individual settings on their own, the tutorials here walk through a complete task from start to finish — which options to enable, in what order, and what to keep in mind so the result behaves as expected.

They cover recurring administrative tasks such as controlling **access and authentication** (guest access, two-factor authentication), configuring **exports and metadata mappings**, setting up **previews and captions**, enabling **text search in files**, and **sharing collections** with other users or external recipients.

Each tutorial is self-contained and points to the relevant reference pages — for example the [Base Configuration](../../../for-administrators/readme/README.md) and [Permissions](../../../for-administrators/permissions/README.md) — where the individual settings are documented in full.

Pick a tutorial from the list to get started.
