# Download & Export

This section provides an overview of how **files and data** can be downloaded or exported from fylr. Depending on your permissions and configuration, you can download **individual files**, **multiple files**, or **entire collections**, as well as export **metadata and structured data** for further processing or external use.

The following articles explain the available **download and export options**, how **renditions, metadata mappings, and file naming** are applied, and how permissions influence what content is accessible.&#x20;

Whether you want to retrieve a single file, distribute selected assets, or export large datasets, these articles guide you through the available options to ensure secure and consistent results.
