# Location Defaults

fylr can store files in more than one **storage location** (configured on the server). Here you choose which location is used by default for each kind of file.

### Originals

The storage location for uploaded original files.

### Versions

The storage location for the versions (renditions) the file worker generates from the originals, such as previews and other derivatives.

### Backups

The storage location used for backups.
